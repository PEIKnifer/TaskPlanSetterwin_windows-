# 用户操作手册

## 1. 主界面说明

- 左侧：任务计划文件夹树
- 右上：任务列表（任务名 / 状态 / 下次运行）
- 右下：任务编辑区（启用、程序路径、参数、起始于）

## 2. 浏览与选择任务

1. 在左侧树中选择目标文件夹。
2. 在右侧列表中点击任务条目。
3. 任务详情会自动加载到编辑区。

## 3. 修改任务

1. 按需修改“启用”“程序路径”“参数”“起始于”。
2. 可通过“浏览...”选择程序路径。
3. 点击“保存”提交修改。

## 4. 新建任务

1. 点击“新建任务”。
2. 输入任务名与程序参数信息。
3. 如需更多配置，点击“高级...”设置启动用户、触发方式等。
4. 点击“创建”。

## 5. 删除任务

1. 在列表中选中任务。
2. 点击“删除任务”并确认。

## 6. 高级设置

点击主界面“高级...”可调整：

- 启动用户、启动方式、权限级别
- 启动时机与触发参数
- 电池策略、并发策略、隐藏任务等

> 已有任务默认读取当前配置；新建任务默认加载推荐配置。

## 7. 测试启动（推荐）

1. 选中任务。
2. 点击“测试启动”。
3. 工具会立即触发任务运行请求，用于快速验证配置是否正确。

## 8. 常见问题

### Q1: 保存/删除/测试启动失败

可能是权限不足。请尝试“以管理员身份运行”应用后重试。

### Q2: 测试启动后未见效果

- 检查任务动作路径是否正确
- 检查触发用户上下文与权限
- 在系统“任务计划程序”查看最后运行结果与历史日志

### Q3: 构建时提示 exe 被占用

先关闭正在运行的 `TaskPlanSetter.exe` 再构建。
